PROYECTO WORKSPACE
=====================
REQUISITOS:
-Node.js instalado
-MariaDB instalado
-Postman

INSTALACION:
-Clonar repositorio

INSTALAR TODAS LAS DEPENDENCIAS:
- npm install

CORROBORAR AUNTENTICACION Y JWT :
- Con postman en : http://localhost:3000/login

BASE DE DATOS:
-Codigo de la base en ./backend/ecommerce.sql
-Conexion con la base de datos en .backend/server.js(tambien credenciales de acceso)
